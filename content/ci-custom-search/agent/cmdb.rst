CMDB
====

After installation of the package a new menu item will be available in the *CMDB* menu in the agent interface.

.. note::

   In order to grant users access to the *CMDB* menu, you need to add them as member to the group *itsm-configitem*.

   The menu items that were added by this package will be visible after you log-in to the system again.
