Customer Activity Widget
========================

This feature add-on provides a dashboard widget that allows agents to define a set of customers to automatically query. It displays a table showing the related number of open, locked and escalated tickets. Drill down to specific tickets by simply clicking on one of the numbers shown per customer.

Benefits
   - Shows the current distribution of tickets at a single glance.
   - Tickets can be selected from the overview using just one click.
   - Facilitates ticket prioritization.

Target Groups
   - IT service providers
   - Finance
   - Sales
   - and many more

Available in Service Package
   - SILVER

Package Name in OTRS Package Manager
   - OTRSCustomerActivityWidget

   Not compatible with the following feature add-ons:
