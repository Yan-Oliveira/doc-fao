Dynamic Field Calculation
=========================

This feature gives a configuration interface for creating or changing dynamic fields of type calculation. Simple mathematical operations and OTRS tags like ``<OTRS_TICKET_AccountedTime>`` can be used.

Target Groups
   - Finance and accounting
   - Project management
   - Security management

Available in Service Package
   - GOLD

Package Name in OTRS Package Manager
   - OTRSDynamicFieldCalculation
